# Spec: left-panel-settings

## 한 줄 정의
사용자가 왼쪽 패널의 Settings 버튼을 클릭하면 설정 모달이 열리고, 기존에 열려 있던 다른 모달은 닫힌다.

## 진입점
- 버튼: 왼쪽 캐릭터 패널 하단의 Settings 버튼 클릭

## 사용자 흐름
1. 사용자가 왼쪽 패널의 Settings 버튼을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 설정 모달이 열리고 설정 컨트롤이 표시된다.
4. 설정을 변경하고 Done을 클릭하면 값이 저장되고 UI가 갱신된다.

## 성공 조건
- [ ] Settings 클릭 시 설정 모달이 열린다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌다.
- [ ] Done 클릭 시 설정이 저장되고 UI에 즉시 반영된다.

## 실패 / 취소 흐름
- X 또는 Escape로 설정 모달 닫기 → 이미 확인되지 않은 설정은 저장되지 않는다.

## 경계 / 예외
- 다른 모달이 열려 있는 상태에서 Settings 클릭 → 이전 모달이 닫히고 설정 모달만 열린다.

## 범위 외
- 설정 모달 내부의 개별 설정 항목.
- 헤더 설정 아이콘과의 동작 일치성.
