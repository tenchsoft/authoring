# Spec: modal-close-x

## 한 줄 정의
사용자가 모달의 X 버튼을 클릭하면 현재 열려 있는 모달이 즉시 닫히고, 임시 변경 사항은 저장되지 않는다.

## 진입점
- 버튼: 모달 오른쪽 상단의 X 버튼 클릭
- 단축키: Escape (모달 열려 있을 때)

## 사용자 흐름
1. 사용자가 모달이 열려 있는 상태에서 X 버튼을 클릭한다.
2. 모달이 즉시 닫힌다.
3. 임시로 입력한 데이터는 저장되지 않는다.

## 성공 조건
- [ ] X 버튼 클릭 시 모달이 닫힌다.
- [ ] 임시 변경 사항이 커밋되지 않는다.
- [ ] Escape 키도 동일한 닫기 동작을 수행한다.

## 실패 / 취소 흐름
- 모달이 열려 있지 않을 때 X 클릭 → 아무 일도 일어나지 않는다.

## 경계 / 예외
- 캐릭터 에디터에서 X 클릭 → 캐릭터가 추가되지 않는다.
- 페르소나 에디터에서 X 클릭 → 페르소나 데이터가 변경되지 않는다.

## 범위 외
- Done 버튼 동작.
- 모달 외부 클릭으로 닫기 동작.
