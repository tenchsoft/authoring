# Spec: template-picker

## 한 줄 정의
사용자가 Template 버튼을 클릭하면 템플릿 선택 모달이 열리고, 템플릿을 선택하면 해당 캐릭터/스토리 설정이 적용된다.

## 진입점
- 버튼: 왼쪽 캐릭터 패널의 Template 버튼 클릭

## 사용자 흐름
1. 사용자가 Template 버튼을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 템플릿 선택 모달이 열리고 템플릿 행이 표시된다.
4. 사용자가 템플릿을 선택하고 확인하면 해당 캐릭터/스토리 설정이 적용된다.

## 성공 조건
- [ ] Template 클릭 시 템플릿 선택 모달이 열린다.
- [ ] 템플릿 선택 후 확인 시 해당 설정이 적용된다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌다.

## 실패 / 취소 흐름
- 선택하지 않고 모달 닫기 → 기존 캐릭터와 페르소나가 변경되지 않는다.

## 경계 / 예외
- 다른 모달이 열려 있는 상태에서 Template 클릭 → 이전 모달이 닫히고 템플릿 선택 모달만 열린다.

## 범위 외
- 템플릿 목록의 세부 내용.
- 템플릿 적용 후 개별 캐릭터/스토리 데이터 변경.
