# Spec: character-row

## 한 줄 정의
사용자가 캐릭터 행을 클릭하면 해당 캐릭터가 활성화되고, 선택 하이라이트가 이동하며, 오른쪽 패널의 캐릭터 상세 정보가 갱신된다.

## 진입점
- 클릭: 왼쪽 캐릭터 목록의 캐릭터 행 클릭

## 사용자 흐름
1. 사용자가 캐릭터 목록에서 특정 캐릭터 행을 클릭한다.
2. 해당 행에 선택 스타일(활성 배경/레일)이 적용된다.
3. 이전에 선택된 행의 하이라이트가 제거된다.
4. 오른쪽 패널에 선택된 캐릭터의 상세 정보가 표시된다.
5. "Selected: <캐릭터 이름>" 토스트가 나타난다.

## 성공 조건
- [ ] 캐릭터 행 클릭 시 해당 행에 선택 스타일이 적용된다.
- [ ] 오른쪽 패널의 활성 캐릭터 상세 정보가 갱신된다.
- [ ] "Selected" 토스트가 나타난다.

## 실패 / 취소 흐름
- 필터링 후 캐릭터 클릭 → 올바른 캐릭터가 선택된다.

## 경계 / 예외
- 목록이 변경된 후 이전 행 클릭 → 앱이 중단되지 않는다.
- 메시지 전송 후 캐릭터 선택 변경 → 응답 화자가 새 캐릭터로 변경된다.
- 세션 복원 시 → 저장된 캐릭터에 하이라이트가 복원된다.

## 범위 외
- 캐릭터 생성/편집.
- 캐릭터 삭제 후 선택 상태 처리.
