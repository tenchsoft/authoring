# Spec: automatic-active-mode-content-render

## 한 줄 정의
중앙 패널이 현재 모드에 따라 자동으로 Chat, Novel, Interactive, Script 콘텐츠를 렌더링한다.

## 진입점
- 자동: 모드 전환 시 자동 발동

## 사용자 흐름
1. 사용자가 모드를 전환한다 (예: Chat → Novel).
2. 중앙 패널이 새 모드에 맞는 형식으로 자동 전환된다.
3. 추가 사용자 조작 없이 콘텐츠가 올바른 형식으로 표시된다.

## 성공 조건
- [ ] Chat 모드 → 채팅 메시지 또는 빈 프롬프트가 렌더링된다.
- [ ] Novel 모드 → 메시지가 산문 형식으로 렌더링된다.
- [ ] Interactive 모드 → 스토리 블록과 선택지가 렌더링된다.
- [ ] Script 모드 → 메시지가 화자 라벨과 함께 렌더링된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 빈 세션에서 모드 전환 → 각 모드의 빈 상태가 올바르게 표시된다.
- 메시지가 많은 상태에서 모드 전환 → 레이아웃이 읽기 가능하게 유지된다.

## 범위 외
- 각 모드의 개별 렌더링 세부사항.
- 메시지 전송 동작.
