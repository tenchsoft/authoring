# Spec: header-settings-icon

## 한 줄 정의
사용자가 헤더 설정 아이콘을 클릭하면 설정 모달이 열리고 기존에 열려 있던 다른 모달은 닫힌다.

## 진입점
- 버튼: 헤더의 설정(톱니바퀴) 아이콘 버튼 클릭

## 사용자 흐름
1. 사용자가 헤더의 설정 아이콘을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 설정 모달이 열리고 설정 컨트롤이 표시된다.

## 성공 조건
- [ ] 설정 아이콘 클릭 시 설정 모달이 열린다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌 후 설정 모달만 표시된다.

## 실패 / 취소 흐름
- Escape 키 → 설정 모달이 닫힌다.
- 설정 반복 클릭 → 설정 모달은 하나만 존재한다.

## 경계 / 예외
- 캐릭터 에디터가 열려 있는 상태에서 설정 클릭 → 캐릭터 에디터가 닫히고 설정 모달만 열린다.

## 범위 외
- 설정 모달 내부의 개별 설정 항목은 별도 spec.
- 설정 저장/영속화 동작.
