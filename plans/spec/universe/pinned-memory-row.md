# Spec: pinned-memory-row

## 한 줄 정의
사용자가 고정 메모리 행을 클릭하면 해당 메모리가 검사 또는 편집을 위해 열리고, 선택 스타일이 적용되며, 채팅 메시지는 전송되지 않는다.

## 진입점
- 클릭: 오른쪽 페르소나 패널의 고정 메모리 행 클릭

## 사용자 흐름
1. 사용자가 고정 메모리 행을 클릭한다.
2. 해당 행에 선택 스타일이 적용된다.
3. 메모리 상세/편집 상태가 열린다.
4. 채팅 메시지는 전송되지 않는다.

## 성공 조건
- [ ] 메모리 행 클릭 시 해당 행에 선택 스타일이 적용된다.
- [ ] 메모리 상세가 열리거나 행이 하이라이트된다.
- [ ] 채팅 메시지가 전송되지 않는다.

## 실패 / 취소 흐름
- 활성 캐릭터가 없는 상태에서 클릭 → 빈 메모리 상태가 표시되며 앱이 중단되지 않는다.

## 경계 / 예외
- 다른 메모리 행 클릭 → 선택이 새 행으로 이동한다.
- 메모리 텍스트 편집 후 저장 → 오른쪽 패널에 갱신된 메모리 내용이 표시된다.

## 범위 외
- 메모리 편집 모달 내부 세부사항.
- 메모리 데이터 영속화.
