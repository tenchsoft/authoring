# Spec: modal-backdrop-dismiss

## 한 줄 정의
사용자가 모달 외부 영역을 클릭하면 명시적 닫기 규칙에 따라 모달이 닫히거나 유지되며, Done 동작은 실행되지 않는다.

## 진입점
- 클릭: 모달 콘텐츠 외부의 배경 영역 클릭

## 사용자 흐름
1. 사용자가 모달이 열려 있는 상태에서 모달 콘텐츠 외부를 클릭한다.
2. 닫기 규칙에 따라 모달이 닫히거나 유지된다.
3. 닫히는 경우 임시 변경 사항은 커밋되지 않는다.

## 성공 조건
- [ ] 모달 외부 클릭 시 닫기 규칙이 적용된다.
- [ ] 임시 데이터가 커밋되지 않는다.
- [ ] 모달 콘텐츠 내부 클릭 → 배경 닫기 동작이 실행되지 않는다.

## 실패 / 취소 흐름
- 모달이 열려 있지 않을 때 → 배경 UI가 정상적으로 클릭을 수신한다.

## 경계 / 예외
- 캐릭터 에디터 외부 클릭 → 저장되지 않은 임시 변경이 커밋되지 않는다.
- 설정에서 값 변경 후 외부 클릭 → 폐기 확인 또는 모달 유지.

## 범위 외
- 각 모달 유형별 닫기/유지 정책 세부사항.
- X 버튼 및 Done 버튼 동작.
