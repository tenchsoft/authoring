# Spec: new-character

## 한 줄 정의
사용자가 + New 버튼을 클릭하면 캐릭터 에디터 모달이 열리고, 확인 전까지 새 캐릭터는 생성되지 않는다.

## 진입점
- 버튼: 왼쪽 캐릭터 패널의 + New 버튼 클릭

## 사용자 흐름
1. 사용자가 + New 버튼을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 캐릭터 에디터 모달이 빈/기본 필드 상태로 열린다.
4. 사용자가 필드를 입력하고 Done을 클릭하면 새 캐릭터 행이 왼쪽 패널에 나타난다.

## 성공 조건
- [ ] + New 클릭 시 캐릭터 에디터 모달이 빈 필드로 열린다.
- [ ] 유효한 데이터로 Done 클릭 시 새 캐릭터 행이 목록에 추가된다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌다.

## 실패 / 취소 흐름
- 모달을 저장하지 않고 닫기(X 또는 Escape) → 새 캐릭터가 추가되지 않는다.
- 필수 필드가 비어 있는 상태에서 Done → 모달이 열린 상태로 유지되고 유효성 메시지가 표시된다.

## 경계 / 예외
- 설정 모달이 열려 있는 상태에서 + New 클릭 → 설정이 먼저 닫히고 캐릭터 에디터가 열린다.

## 범위 외
- 캐릭터 에디터 모달 내부의 개별 필드.
- 캐릭터 데이터 영속화 세부사항.
