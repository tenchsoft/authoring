# Spec: automatic-prompt-preview-width

## 한 줄 정의
프롬프트 미리보기가 열려 있을 때 오른쪽 패널이 자동으로 넓어지고 중앙 패널이 줄어들어 미리보기 영역을 확보한다.

## 진입점
- 자동: 프롬프트 미리보기 열기/닫기, 창 크기 변경 시 자동 발동

## 사용자 흐름
1. 프롬프트 미리보기가 열리면 오른쪽 패널이 미리보기 너비로 확장된다.
2. 중앙 패널이 그에 맞게 축소된다.
3. 프롬프트 미리보기가 닫히면 오른쪽 패널이 일반 너비로 돌아온다.
4. 창 크기 변경 시 중앙과 오른쪽 패널 경계가 유효하게 유지된다.

## 성공 조건
- [ ] 프롬프트 미리보기 열기 시 오른쪽 패널이 확장된다.
- [ ] 프롬프트 미리보기 닫기 시 오른쪽 패널이 일반 너비로 돌아온다.
- [ ] 창 크기 변경 시 패널 경계가 유효하게 유지된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 매우 작은 창 → 중앙 패널 너비가 최소값으로 제한된다.
- 미리보기 열린 상태에서 작성기 사용 → 작성기 너비가 중앙 패널에 맞춰진다.

## 범위 외
- 프롬프트 미리보기 콘텐츠.
- 패널 경계 드래그 조정.
