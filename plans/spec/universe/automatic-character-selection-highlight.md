# Spec: automatic-character-selection-highlight

## 한 줄 정의
활성 캐릭터 행에 자동으로 선택 스타일이 적용되고, 오른쪽 패널의 활성 캐릭터 상세 정보가 갱신된다.

## 진입점
- 자동: 캐릭터 선택, 목록 변경, 세션 복원 시 자동 발동

## 사용자 흐름
1. 사용자가 캐릭터를 선택하거나 세션이 복원된다.
2. 선택된 캐릭터 행에 활성 배경/레일 스타일이 자동으로 적용된다.
3. 이전 선택 행의 스타일이 제거된다.
4. 오른쪽 패널에 선택된 캐릭터의 상세 정보가 표시된다.

## 성공 조건
- [ ] 캐릭터 선택 시 해당 행에 선택 스타일이 적용된다.
- [ ] 다른 캐릭터 선택 시 하이라이트가 이동한다.
- [ ] 오른쪽 패널의 활성 캐릭터 상세가 갱신된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 캐릭터 삭제/필터 후 → 선택된 식별자가 유효하게 유지되거나 안전하게 조정된다.
- 세션 복원 시 → 저장된 캐릭터에 하이라이트가 복원된다.

## 범위 외
- 캐릭터 행 클릭 동작 (별도 spec).
- 캐릭터 상세 정보 내용 세부사항.
