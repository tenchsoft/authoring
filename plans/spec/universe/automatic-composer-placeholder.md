# Spec: automatic-composer-placeholder

## 한 줄 정의
작성기 입력이 비어 있을 때 플레이스홀더 텍스트가 자동으로 표시되고, 텍스트가 있으면 사라진다.

## 진입점
- 자동: 입력, 전송, 백스페이스 시 자동 발동

## 사용자 흐름
1. 빈 작성기에서 "Type a message" 플레이스홀더가 표시된다.
2. 텍스트를 입력하면 플레이스홀더가 사라지고 입력 텍스트가 표시된다.
3. 메시지를 전송하면 입력이 비워지고 플레이스홀더가 돌아온다.
4. 백스페이스로 모든 텍스트를 지우면 플레이스홀더가 돌아온다.

## 성공 조건
- [ ] 빈 입력 상태에서 플레이스홀더가 표시된다.
- [ ] 텍스트 입력 시 플레이스홀더가 사라진다.
- [ ] 메시지 전송 후 플레이스홀더가 돌아온다.
- [ ] 백스페이스로 빈 상태가 되면 플레이스홀더가 돌아온다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 플레이스홀더 텍스트가 저장된 입력 데이터에 포함되지 않는다.

## 범위 외
- 작성기 텍스트 입력 동작.
- 전송 동작.
