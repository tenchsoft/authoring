# Spec: persona-edit

## 한 줄 정의
사용자가 Edit 버튼을 클릭하면 페르소나 에디터 모달이 열리고, 페르소나 초안 필드를 편집할 수 있다.

## 진입점
- 버튼: 오른쪽 페르소나 패널의 Edit 버튼 클릭

## 사용자 흐름
1. 사용자가 오른쪽 패널의 Edit 버튼을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 페르소나 에디터 모달이 현재 페르소나 이름, 소개, 톤으로 열린다.
4. 사용자가 필드를 수정하고 Done을 클릭하면 오른쪽 패널의 페르소나 정보가 갱신된다.

## 성공 조건
- [ ] Edit 클릭 시 페르소나 에디터 모달이 현재 값으로 열린다.
- [ ] Done 클릭 후 오른쪽 패널의 페르소나 정보가 갱신된다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌다.

## 실패 / 취소 흐름
- 저장하지 않고 모달 닫기 → 기존 페르소나 값이 변경되지 않는다.

## 경계 / 예외
- 설정 모달이 열려 있는 상태에서 Edit 클릭 → 설정이 먼저 닫히고 페르소나 에디터가 열린다.

## 범위 외
- 페르소나 에디터 모달 내부의 개별 필드.
- 페르소나 데이터 영속화 세부사항.
