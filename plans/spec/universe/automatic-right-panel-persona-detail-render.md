# Spec: automatic-right-panel-persona-detail-render

## 한 줄 정의
오른쪽 패널이 페르소나, 고정 메모리, 활성 캐릭터 상세 정보를 자동으로 렌더링한다.

## 진입점
- 자동: 페르소나 편집, 캐릭터 선택, 메모리 변경, 세션 변경 시 자동 발동

## 사용자 흐름
1. 사용자가 페르소나를 편집하고 저장한다 → 이름, 톤, 소개가 오른쪽 패널에 갱신된다.
2. 캐릭터를 선택한다 → 활성 캐릭터 설명, 점수, 태그가 갱신된다.
3. 고정 메모리를 갱신한다 → 메모리 행이 새로고침된다.
4. 활성 캐릭터가 없으면 → 안전한 빈 상태가 표시된다.

## 성공 조건
- [ ] 페르소나 편집 후 오른쪽 패널의 페르소나 필드가 갱신된다.
- [ ] 캐릭터 선택 후 활성 캐릭터 데이터가 갱신된다.
- [ ] 메모리 갱신 후 메모리 행이 새로고침된다.
- [ ] 활성 캐릭터 없을 때 빈 상태가 표시된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 세션 전환 시 → 페르소나와 캐릭터 정보가 새 세션에 맞게 갱신된다.

## 범위 외
- 페르소나 편집 모달 동작.
- 캐릭터 선택 동작.
