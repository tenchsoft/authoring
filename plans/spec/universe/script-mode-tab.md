# Spec: script-mode-tab

## 한 줄 정의
사용자가 Script 모드 탭을 클릭하면 중앙 패널이 대본 형식 렌더링으로 전환되고 Script 탭이 활성 상태가 된다.

## 진입점
- 탭: 헤더의 Script 모드 탭 클릭

## 사용자 흐름
1. 사용자가 헤더의 Script 탭을 클릭한다.
2. 중앙 패널이 화자 라벨이 있는 대본 형식으로 전환된다.
3. Script 탭에 활성 스타일이 적용된다.
4. "Mode: Script" 토스트가 나타난다.

## 성공 조건
- [ ] Script 탭 클릭 시 중앙 패널이 대본 형식으로 전환된다.
- [ ] Script 탭에 활성 스타일이 표시된다.
- [ ] "Mode: Script" 토스트가 나타난다.

## 실패 / 취소 흐름
- 모달이 열려 있는 상태에서 탭 클릭 → 모달이 가려진 UI를 통해 모드가 의도치 않게 변경되지 않는다.

## 경계 / 예외
- 이미 Script 모드일 때 다시 클릭 → 중복 데이터 변경 없이 현재 상태 유지.
- 다른 모드에서 Script로 전환 후 메시지 전송 → 대본 형식으로 메시지가 표시된다.

## 범위 외
- 다른 모드(Chat, Novel, Interactive) 전환은 각각의 별도 spec.
- 모드 전환 시 기존 메시지, 캐릭터, 모달 상태 보존 정책.
