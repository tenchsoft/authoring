# Spec: character-search-field

## 한 줄 정의
사용자가 캐릭터 검색 필드에 텍스트를 입력하면 캐릭터 목록이 즉시 필터링된다.

## 진입점
- 입력 필드: 왼쪽 캐릭터 패널의 검색 필드 활성화

## 사용자 흐름
1. 사용자가 검색 필드를 클릭하여 포커스를 얻는다.
2. 텍스트를 입력하면 캐릭터 목록이 입력에 맞게 즉시 필터링된다.
3. 검색어를 지우면 전체 캐릭터 목록이 복원된다.

## 성공 조건
- [ ] 검색 필드 클릭 시 포커스를 받는다.
- [ ] 텍스트 입력 시 일치하는 캐릭터만 목록에 표시된다.
- [ ] 검색어 삭제 시 전체 캐릭터 목록이 복원된다.

## 실패 / 취소 흐름
- 다른 컨트롤 클릭 → 검색 포커스가 해제되고 입력 텍스트가 채팅 작성기로 전달되지 않는다.

## 경계 / 예외
- 대소문자 혼합 입력 → 대소문자 구분 없이 검색된다.
- 빈 캐릭터 목록에서 검색 → 빈 상태가 유지된다.

## 범위 외
- 캐릭터 생성/편집 동작.
- 검색 결과 캐릭터 선택 동작은 캐릭터 행 제어 spec 참조.
