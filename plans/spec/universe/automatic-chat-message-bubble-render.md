# Spec: automatic-chat-message-bubble-render

## 한 줄 정의
Chat 모드에서 메시지가 변경될 때마다 자동으로 사용자 및 캐릭터 버블이 렌더링된다.

## 진입점
- 자동: 메시지 전송, 세션 로드 시 자동 발동

## 사용자 흐름
1. 사용자가 메시지를 보내거나 세션을 로드한다.
2. 사용자 메시지는 오른쪽 정렬 버블로 나타난다.
3. 캐릭터 응답은 왼쪽 정렬 버블로 나타난다.
4. 빈 세션에서는 빈 프롬프트가 표시된다.

## 성공 조건
- [ ] 사용자 메시지가 오른쪽 정렬 버블로 렌더링된다.
- [ ] 캐릭터 응답이 왼쪽 정렬 버블로 렌더링된다.
- [ ] 빈 세션에서 빈 프롬프트가 표시된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 메시지가 많을 때 → 수직 레이아웃이 읽기 가능하거나 스크롤 가능하게 유지된다.
- 오래된 버블이 표시되지 않는다.

## 범위 외
- 메시지 전송 동작.
- 다른 모드의 메시지 렌더링.
