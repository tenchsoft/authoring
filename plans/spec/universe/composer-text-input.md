# Spec: composer-text-input

## 한 줄 정의
사용자가 작성기 입력 필드에 텍스트를 입력하면 텍스트가 표시되고, 플레이스홀더가 사라지며, 전송 버튼이나 Enter를 누를 때까지 메시지가 생성되지 않는다.

## 진입점
- 입력 필드: 하단 메시지 작성기의 텍스트 입력 필드에 타이핑

## 사용자 흐름
1. 사용자가 작성기 입력 필드를 클릭하여 포커스를 얻는다.
2. 텍스트를 입력하면 플레이스홀더가 사라지고 입력한 텍스트가 표시된다.
3. Backspace로 마지막 문자를 안전하게 삭제할 수 있다.
4. 전송 버튼이나 Enter를 누를 때까지 텍스트는 메시지로 생성되지 않는다.

## 성공 조건
- [ ] 텍스트 입력 시 플레이스홀더가 사라지고 입력 텍스트가 표시된다.
- [ ] Backspace로 마지막 문자가 안전하게 삭제된다.
- [ ] 전송 전까지 채팅 메시지가 생성되지 않는다.

## 실패 / 취소 흐름
- 모달이나 검색 필드에 포커스가 있는 상태에서 타이핑 → 텍스트가 작성기가 아닌 해당 컨트롤로 전달된다.

## 경계 / 예외
- 멀티바이트 텍스트 입력 → 문자 단위로 안전하게 처리된다.
- 빈 입력 상태에서 Backspace → 앱이 중단되지 않는다.

## 범위 외
- 전송 버튼 동작.
- Enter 단축키 전송 동작.
