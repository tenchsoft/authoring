# Spec: backspace-composer

## 한 줄 정의
사용자가 Backspace 키를 누르면 작성기 입력의 마지막 문자가 삭제되고, 메시지나 선택된 캐릭터는 변경되지 않는다.

## 진입점
- 단축키: Backspace (작성기 포커스 상태)

## 사용자 흐름
1. 사용자가 작성기에 텍스트를 입력한 상태에서 Backspace를 누른다.
2. 마지막 문자가 삭제된다.
3. 메시지나 선택된 캐릭터에는 영향이 없다.

## 성공 조건
- [ ] 텍스트가 있는 상태에서 Backspace → 마지막 문자가 삭제된다.
- [ ] 빈 입력 상태에서 Backspace → 앱이 중단되지 않는다.

## 실패 / 취소 흐름
- 모달 입력 필드에 포커스가 있는 상태에서 Backspace → 모달 임시 텍스트가 변경되며 작성기 입력은 변경되지 않는다.

## 경계 / 예외
- 멀티바이트 텍스트에서 Backspace → 문자 단위로 안전하게 삭제된다.

## 범위 외
- 전송 동작.
- 다른 단축키와의 상호작용.
