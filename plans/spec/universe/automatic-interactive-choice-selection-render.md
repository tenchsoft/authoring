# Spec: automatic-interactive-choice-selection-render

## 한 줄 정의
인터랙티브 모드에서 선택한 선택지의 시각적 상태가 자동으로 갱신된다.

## 진입점
- 자동: 인터랙티브 선택지 선택, 모드 전환 시 자동 발동

## 사용자 흐름
1. 사용자가 인터랙티브 선택지를 선택한다.
2. 해당 선택지에 선택 스타일이 자동으로 적용된다.
3. 이전 선택지의 스타일이 제거된다.
4. 다른 모드로 전환 후 돌아와도 선택 상태가 유지된다.

## 성공 조건
- [ ] 선택지 선택 시 해당 행에 선택 스타일이 적용된다.
- [ ] 다른 선택지 선택 시 이전 선택이 해제되고 새 선택이 하이라이트된다.
- [ ] 모드 전환 후 복귀 시 선택 상태가 유지된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 선택 없는 블록 로드 → 모든 선택지가 미선택 상태로 렌더링된다.

## 범위 외
- 인터랙티브 선택지 클릭 동작 (별도 spec).
- 스토리 상태 진행.
