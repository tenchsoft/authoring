# Spec: enter-send

## 한 줄 정의
사용자가 Enter 키를 누르면 작성기의 현재 입력 텍스트가 Send 버튼과 동일한 경로로 전송된다.

## 진입점
- 단축키: Enter (작성기 포커스 상태)

## 사용자 흐름
1. 사용자가 작성기에 텍스트를 입력한다.
2. Enter 키를 누른다.
3. 입력 텍스트가 사용자 메시지로 전송되고 캐릭터 응답이 추가된다.
4. 작성기가 비워진다.

## 성공 조건
- [ ] 비어있지 않은 입력 상태에서 Enter → 메시지가 전송된다.
- [ ] 빈 입력 상태에서 Enter → 빈 메시지가 생성되지 않는다.

## 실패 / 취소 흐름
- 모달 폼 필드에 포커스가 있는 상태에서 Enter → 모달 필드가 확인되며 채팅이 전송되지 않는다.

## 경계 / 예외
- 전송 직후 반복 Enter → 중복 빈 메시지가 생성되지 않는다.
- 검색 필드 포커스 상태에서 Enter → 검색이 실행되며 채팅이 전송되지 않는다.

## 범위 외
- Send 버튼 동작.
- Shift+Enter로 줄바꿈 지원 여부.
