# Spec: automatic-local-runtime-status-render

## 한 줄 정의
헤더에 현재 런타임 상태가 자동으로 표시되어 사용자가 로컬 Engine/런타임이 온라인인지 확인할 수 있다.

## 진입점
- 자동: 런타임 연결/연결 해제, 설정 변경 시 자동 발동

## 사용자 흐름
1. 런타임이 온라인이면 → 헤더에 온라인/로컬 런타임 상태가 표시된다.
2. 런타임 연결이 끊기면 → 사용자 조작 없이 상태가 변경된다.
3. 외부 제공자가 선택되면 → 해당 제공자 상태가 반영된다.
4. 설정에서 런타임 모드를 변경하면 → 헤더 상태가 즉시 갱신된다.

## 성공 조건
- [ ] 런타임 온라인 시 온라인 상태가 헤더에 표시된다.
- [ ] 런타임 연결 해제 시 상태가 자동으로 변경된다.
- [ ] 외부 제공자 선택 시 해당 상태가 반영된다.
- [ ] 설정 변경 시 상태가 즉시 갱신된다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 네트워크 엔드포인트가 불필요하게 노출되지 않는다.

## 범위 외
- 설정 모달 내부의 런타임 설정.
- Engine IPC 통신 세부사항.
