# Spec: automatic-toast-display

## 한 줄 정의
모드 전환, 캐릭터 선택, 선택지 선택 등의 동작 시 자동으로 토스트가 하단 근처에 나타난다.

## 진입점
- 자동: 모드 전환, 캐릭터 선택, 인터랙티브 선택 시 자동 발동

## 사용자 흐름
1. 사용자가 모드를 전환한다 → "Mode: <모드>" 토스트가 나타난다.
2. 사용자가 캐릭터를 선택한다 → "Selected: <캐릭터>" 토스트가 나타난다.
3. 사용자가 인터랙티브 선택지를 선택한다 → "You chose: <선택지>" 토스트가 나타난다.
4. 설정된 시간이 지나면 토스트가 자동으로 사라진다.

## 성공 조건
- [ ] 모드 전환 시 해당 모드 토스트가 나타난다.
- [ ] 캐릭터 선택 시 선택 캐릭터 토스트가 나타난다.
- [ ] 인터랙티브 선택 시 선택 토스트가 나타난다.
- [ ] 설정된 시간 후 토스트가 자동으로 사라진다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 연속 동작 시 → 최신 토스트가 이전 것을 대체한다.

## 범위 외
- 토스트 지속 시간 설정.
- 토스트 위치 세부사항.
