# Spec: automatic-modal-exclusivity

## 한 줄 정의
하나의 모달을 열면 다른 모든 모달이 자동으로 닫혀 한 번에 하나의 모달만 표시된다.

## 진입점
- 자동: 모달 열기 동작 시 자동 발동

## 사용자 흐름
1. 사용자가 한 모달을 연다 (예: 캐릭터 에디터).
2. 다른 모달을 열면 (예: 설정) 이전 모달이 자동으로 닫히고 새 모달만 표시된다.
3. 현재 모달을 닫으면 모든 모달이 닫힌다.
4. 같은 모달을 두 번 열어도 하나만 표시된다.

## 성공 조건
- [ ] 새 모달 열기 시 기존 모달이 닫힌다.
- [ ] 한 번에 하나의 모달만 표시된다.
- [ ] 모달 닫기 시 모든 모달이 닫힌다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 캐릭터 에디터 → 설정 전환 시 캐릭터 에디터만 닫히고 설정만 열린다.
- 템플릿 선택 → 세션 전환 시 템플릿 선택만 닫히고 세션만 열린다.

## 범위 외
- 개별 모달의 내용.
- 모달 닫기 버튼, 배경 클릭 동작.
