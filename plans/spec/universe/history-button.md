# Spec: history-button

## 한 줄 정의
사용자가 History 버튼을 클릭하면 대화 세션 모달이 열리고, 기존 세션을 선택하여 복원할 수 있다.

## 진입점
- 버튼: 왼쪽 캐릭터 패널 하단의 History 버튼 클릭

## 사용자 흐름
1. 사용자가 History 버튼을 클릭한다.
2. 기존에 열려 있던 모달이 닫힌다.
3. 대화 세션 모달이 열리고 기존 세션 행이 표시된다.
4. 사용자가 세션을 선택하면 해당 세션의 메시지와 활성 캐릭터가 복원된다.

## 성공 조건
- [ ] History 클릭 시 대화 세션 모달이 열린다.
- [ ] 세션 선택 시 해당 세션의 메시지와 활성 캐릭터가 복원된다.
- [ ] 다른 모달이 열려 있으면 먼저 닫힌다.

## 실패 / 취소 흐름
- 세션 선택 없이 모달 닫기 → 현재 메시지가 변경되지 않는다.

## 경계 / 예외
- 템플릿 선택 모달이 열려 있는 상태에서 History 클릭 → 템플릿 모달이 먼저 닫힌다.

## 범위 외
- 세션 데이터 영속화 세부사항.
- 세션 삭제/이름 변경.
