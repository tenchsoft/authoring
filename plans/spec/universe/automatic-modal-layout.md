# Spec: automatic-modal-layout

## 한 줄 정의
모달 크기와 위치가 창 크기에 맞게 자동으로 조정되며, 항상 중앙에 배치되고 사용 가능하게 유지된다.

## 진입점
- 자동: 모달 열기, 창 크기 변경 시 자동 발동

## 사용자 흐름
1. 사용자가 모달을 연다.
2. 모달이 창 중앙에 적절한 크기로 표시된다.
3. 창 크기를 변경하면 모달이 자동으로 재배치된다.
4. 닫기 버튼이 항상 접근 가능하게 유지된다.
5. 배경이 모달 주변의 전체 창을 덮는다.

## 성공 조건
- [ ] 넓은 데스크톱에서 모달이 정상 너비로 표시된다.
- [ ] 작은 창에서 모달 너비/높이가 가시 영역 내로 제한된다.
- [ ] 모달 열린 상태에서 창 크기 변경 → 모달이 중앙 재배치된다.
- [ ] 배경이 전체 창을 덮는다.

## 실패 / 취소 흐름
- 해당 없음 (자동 동작).

## 경계 / 예외
- 매우 작은 창 → 모달이 최소 지원 크기로 제한된다.

## 범위 외
- 모달 콘텐츠 세부사항.
- 모달 배경 닫기 동작.
